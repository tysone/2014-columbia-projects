Making Inline Audio Easy and Seamless

If you want to change how a playing clip is styled, you can implement your own progress callback. The callback gets two arguments, the DOM element which was clicked to play a clip, and an integer percentage of the amount of the clip which has been played at the time it was called. You tell Soundcite about your alternative callback using the SOUNDCITE_CONFIG variable somewhere in a <script> tag on your page. This example uses a different color scheme. This page also uses custom CSS to override the default style of a loaded clip.

Examples

<style type="text/css" media="screen">
  .soundcite-loaded {
    background: rgba(195,69,40,.1)
  }
</style>
          
          
<script type="text/javascript">         
  var SOUNDCITE_CONFIG = {
    update_playing_element: function(el, percentage) {
      $(el).css({
        'background' : '-webkit-linear-gradient(left, rgba(195,69,40,.25)' 
          + percentage + '%, rgba(195,69,40,.10)' 
          + (percentage + 1) + '%)',
        'background' : 'linear-gradient(to right, rgba(195,69,40,.25)' 
          + percentage + '%, rgba(195,69,40,.10)' 
          + (percentage + 1) + '%)'
    });
  }
}
</script>
